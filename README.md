# RestoPOS

RestoPOS adalah aplikasi point of sale (POS) berbasis mobile yang mendukung dua jenis pengguna, yaitu customer dan kasir.

Kasir dapat mengelola pesanan, menambahkan menu, menghapus menu, dan mengubah status pesanan. 

Customer dapat melihat menu, menambahkan menu ke keranjang, menghapus menu dari keranjang, dan memesan menu.

Page Kasir:
- Login/Register
- Dashboard
- List Orders

Page Customer:
- Login/Register
- Dashboard
- Cart
- Checkout
- History Orders

Auth Firebase: Email & Password, Anonymous(guest account).
